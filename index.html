Welcome to EngiPay Wallet SDK Docs

The EngiPay Wallet is a non-custodial digital wallet built on StarkNet, offering secure and efficient transactions. Our SDKs and APIs are designed to provide developers with the tools needed to seamlessly integrate EngiPay's features into their applications.

Note: This reference guide provides a comprehensive overview of the libraries and tools you can use to build on the StarkNet ecosystem, including specific SDKs for payments, DeFi, and wallet integration.

Core Features

StarkNet L2 Scalability

Leverage StarkNet's Layer 2 scalability solutions for low-cost and high-speed transactions. Our core library, built on StarkNet.js, simplifies interaction with StarkNet contracts and accounts.

Multi-chain Payments

The EngiPay Wallet supports multi-chain payments via integrations with leading protocols, allowing users to make transactions across different networks from a single interface.

DeFi & Trading

Connect to various DeFi protocols for yield farming, staking, and asset swapping. Our SDKs simplify the process of aggregating liquidity and interacting with complex DeFi contracts.

SDK & API Reference

Core StarkNet & Cairo SDKs

  • StarkNet.js: Core JavaScript library for StarkNet interaction
  • StarkNet Foundry: Testing and deployment toolkit
  • Scarb: Cairo package manager and build system
  • Starkli: Command-line tool for StarkNet operations
  • StarkNet Devnet: Local development network
  • Cairo Book: Official Cairo programming guide
  • OpenZeppelin Contracts for Cairo: Secure smart contract library
  • Cairo by Example: Practical Cairo examples

Core Payment SDKs (Project Specific)

  • Chipi Pay SDK: Next.js integration for instant payments

Traditional Payment Processors

  • Stripe: Complete payments platform
